# TaskFlow Sequence Diagrams

This folder contains Mermaid.js sequence diagrams for all 10 use cases.

## Files

| # | File | Use Case |
|---|------|----------|
| 1 | `01-authentication.mmd` | Role-based session management (Admin/Manager/Employee) |
| 2 | `02-manage-users.mmd` | Admin profile creation & permission auditing |
| 3 | `03-manage-projects.mmd` | Workspace organization & milestone tracking |
| 4 | `04-tasks-kanban.mmd` | Visual state transitions with validation |
| 5 | `05-assign-prioritize.mmd` | Deadlines & outcome-based accountability |
| 6 | `06-time-tracking.mmd` | Actual vs estimated effort analysis |
| 7 | `07-productivity-reports.mmd` | Bias-free performance metrics |
| 8 | `08-bottleneck-detection.mmd` | Workflow accumulation point detection |
| 9 | `09-ml-recommendations.mmd` | Historical analysis & Task of the Day impact |
| 10 | `10-contextual-collaboration.mmd` | Task-linked chat, video rooms, blocked status |

## How to View

1. **VS Code**: Install "Markdown Preview Mermaid Support" extension
2. **Online**: Paste code into [mermaid.live](https://mermaid.live)
3. **GitHub**: Wrap contents in \`\`\`mermaid code blocks
